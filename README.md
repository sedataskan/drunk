# drunk

Günlük içtiğim su miktarını tutan uygulama

## Kurulum

1. Projeyi klonlayın:
   ```bash
   git clone
   ```
2. Proje dizinine gidin:
   ```bash
   cd drunk
   ```
3. Bağımlılıkları yükleyin:
   ```bash
   npm install
   ```
4. Uygulamayı çalıştırın:
   ```bash
   npx expo start
   ```

## Özellikler

- Su tüketimini günlük olarak takip edin.
- İstatistikler ile su tüketim alışkanlıklarınızı analiz edin.
- Kullanıcı dostu arayüz ile kolay kullanım.

## Teknolojiler

- React Native
- Expo
